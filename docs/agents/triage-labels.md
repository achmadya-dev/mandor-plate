# Triage Labels

The skills use one issue type label and three workflow status labels. This file maps those roles to the actual label strings used in this repo's issue tracker.

| Default label     | Label in our tracker | Meaning                                            |
| ----------------- | -------------------- | -------------------------------------------------- |
| `epic`            | `epic`               | Groups child implementation issues for one PR      |
| `needs-triage`    | `needs-triage`       | Maintainer needs to evaluate this issue            |
| `ready-for-agent` | `ready-for-agent`    | Fully specified, ready for the implementation loop |
| `ready-for-human` | `ready-for-human`    | Requires human implementation                      |

When a skill mentions a role (e.g. "apply the AFK-ready triage label"), use the corresponding label string from this table.

Edit the right-hand column to match whatever vocabulary you actually use.
